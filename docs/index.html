How to Contribute

Run the benchmark suite on your machine and submit results:

# Install and run
cargo install --path peregrine-bench
peregrine-bench --json > my_results.json

# Or from the repo
cargo run -p peregrine-bench --release -- --json > my_results.json

# Quick mode (fewer iterations, ~2 min)
cargo run -p peregrine-bench --release -- --quick --json > my_results.json

Then open a PR adding your JSON file to docs/results/.

Built with Peregrine · Benchmarks are median of timed iterations (warmup excluded)
